# El Juego de la Vida — Landing Page

Landing page para **El Juego de la Vida**, salón de eventos y cumpleaños infantiles en Haedo, Zona Oeste.

## Tecnología
Sitio estático en **HTML + CSS + JavaScript** vanilla. Sin frameworks ni build.

```
index.html
css/style.css
js/script.js
assets/img · assets/logo
```

## Desarrollo local
```bash
python -m http.server 5577
# abrir http://localhost:5577
```

## Deploy
Sitio estático — Vercel detecta `index.html` automáticamente (sin configuración).
